========
cusy.cms
========

The main policy package to install the Cusy CMS.


Features
--------


Installation
------------

Install ``cusy.cms`` by adding it to your buildout::

    [buildout]

    ...

    eggs =
        cusy.cms


and then running ``bin/buildout``
